Coding Exercises

Week 1

Exercise 1

Define a function called find_max_value that receives a list of integers as argument and returns its highest value.

# Test it with the following code:
integer_list = [52, 81, 33, 17, 25, 90, 28, 43]
max_value = find_max_value(integer_list)
print(max_value)

Expected output is 90.

Exercise 2

Define a function called find_item_index that takes a list of integers and a value as arguments. If the value is in the list, return its position in the list (i.e. index). If its not, return a feedback message to the user.

# Code for testing:
integer_list = [52, 81, 33, 17, 25, 90, 28, 43]
# 1st test:
value_searched = 17
result = find_item_index(integer_list, value_searched)
print(result)
# 2nd test:
value_searched = 35
result = find_item_index(integer_list, value_searched)
print(result)

Exercise 3

Using Python's utility function input(<message to user>) write a program that accepts a user's string input and outputs its inverse.
Example of expected output:

[In] : anagram
[Out]: margana

Exercise 4

Build a Pyramid!
Create a program that accepts an integer from the user and builds an equivalent number of layers, using #. The idea is to simulate a pyramid.
Example of expected output:

[In] : 8
[Out]:
#
##
###
####
#####
######
#######
########

Exercise 5

Define a function called find_item_index_fast that takes a list of integers and a value as arguments. The objective is the same as in Exercise 2, only now you should aim to accomplish the search in a more efficient way. Independent of the input size (i.e. the list's size).
Tip: Notice this time the example integer_list is ordered...

# Code for testing:
integer_list = [1, 2, 11, 12, 15, 17, 25, 26, 28, 28, 29, 33, 43, 44, 47, 50, 52, 61, 73, 81, 82, 90, 99, 100]
# 1st test:
value_searched = 81
result = find_item_index_fast(integer_list, value_searched)
print(result)
# 2nd test:
value_searched = 30
result = find_item_index_fast(integer_list, value_searched)
print(result)

Exercise 6

Build your very own factorial calculator. The program should take an integer as argument and output its factorial.
Try to find two different solutions to this problem.
